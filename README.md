# SQL Sales and Employee Analysis Project

## Overview
This is a beginner-friendly SQL project created to practice important database concepts such as table creation, data insertion, aggregation, joins, data quality checks, updates, and window functions.

## Tools Used
- SQL
- MySQL

## Files in This Project
- `schema.sql` → creates database, tables, and inserts sample data
- `analysis_queries.sql` → contains sales analysis queries
- `joins_and_data_quality.sql` → contains join queries and missing data checks
- `window_functions.sql` → contains ranking queries using window functions

## Concepts Covered
- CREATE DATABASE
- CREATE TABLE
- INSERT INTO
- SELECT
- WHERE
- GROUP BY
- SUM, COUNT, AVG, MAX, MIN
- UPDATE
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN using UNION
- CROSS JOIN
- ROW_NUMBER()
- RANK()
- DENSE_RANK()

## Business Questions Solved

### Orders Analysis
- What is the total sales per city?
- How many orders came from each city?
- What is the average order value per city?
- What is the highest and lowest order value per city?

### Employee and Department Analysis
- Which employees have valid departments?
- Which employees do not have a matching department?
- Which departments do not have any employees?

### Salary Analysis
- Who has the highest salary?
- How can employees be ranked by salary?
- How can employees be ranked inside each department?

## Learning Outcome
Through this project, I learned how to:
- create and use a database
- create tables and insert records
- summarize data using GROUP BY
- combine multiple tables using joins
- find missing relationships in data
- use window functions for ranking
